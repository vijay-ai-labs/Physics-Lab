# Wave Speed on a String

## What Is This Experiment?
**Definition:** The speed at which a mechanical wave travels down a taut string or wire depends strictly on the physical properties of the medium: the tension pulling the string tight, and the linear mass density (thickness/heaviness) of the string.
**Real-World Example:** Tuning a guitar. When you tighten the tuning peg, you are increasing the *Tension* on the string. This directly increases the wave speed along the string, which increases the frequency (pitch) of the note you hear! Thick bass strings have high *Mass Density*, so waves travel slower, creating lower-pitched notes.

## Formulas Used
The velocity ($v$) of a wave on a string is given by:

$$
v = \sqrt{\frac{T}{\mu}}
$$

Where:
*   $v$ is the wave speed (meters per second, m/s).
*   $T$ is the Tension force applied to the string (Newtons, N).
*   $\mu$ is the Linear Mass Density of the string (kilograms per meter, kg/m).

## Goal of the Experiment
The main objective is to find **What determines wave speed on a string?** to prove that wave speed is completely independent of how hard you pluck it (amplitude) and only depends on string tightness and thickness.

## Simulation Controls
*   **Tension (N):** How tightly the string is pulled. Increasing tension drastically increases the speed of the wave pulse.
*   **Linear Density (kg/m):** How thick and heavy the string is. Thicker strings are sluggish, reducing the wave speed.
*   **Amplitude (m):** How high the initial pulse is. You will notice that changing amplitude *does not* change the horizontal speed of the wave!
*   **Vector/Grid Overlays:**
    *   **Pulse Monitor:** Watch the pulse travel left to right against the background grid to visually confirm the speed changes as you adjust Density and Tension.
